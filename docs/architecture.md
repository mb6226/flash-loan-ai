# معماری سیستم

این سند شرحی کلی از لایه‌ها و کامپوننت‌های سیستم Flash Loan AI را ارائه می‌دهد.

- LAYER 1: Data Ingestion: شامل listeners و mempool_monitor و price_aggregator برای دریافت داده‌های لحظه‌ای از شبکه و DEXها
- LAYER 2: AI Processing: مدل‌ها و پیاده‌سازی inference و training
- LAYER 3: Strategies: الگوریتم‌ها برای یافتن فرصت‌های آربیتراژ، لیکوئیدیشن و تصمیمات ریسک
- LAYER 4: Executors & Automation: اجرای تراکنش‌ها روی شبکه و هماهنگی (scripting, workflows)

سلب‌مسئولیت: این نرم‌افزار صرفا آموزشی است. نباید در شبکه اصلی بدون بررسی‌های امنیتی و حقوقی استفاده شود.
